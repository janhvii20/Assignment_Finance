# 🚀 Finance Dashboard UI

A modern and interactive **Finance Dashboard** built using React.  
This application helps users track financial activity, visualize spending patterns, and manage transactions with a clean and intuitive UI.

---

## 🌐 Live Demo

The application is deployed and available at:  
👉 https://assignment-finance.vercel.app

---

## 📂 Repository

GitHub Repository:  
👉 https://github.com/janhvii20/Assignment_Finance

---

## ⚙️ Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Charts:** Recharts
- **State Management:** Context API + Hooks
- **Storage:** localStorage

---

## 📊 Features

### ## Dashboard Overview

- Displays **Total Balance, Income, and Expenses**
- Balance trend visualization using charts
- Spending breakdown using category-based pie chart

---

### ## Transactions Management

- View all transactions in a structured table
- Search transactions by title or category
- Filter transactions by type (Income / Expense)
- Sort transactions by date and amount

---

### ## Role-Based UI

- **Viewer Mode:** Can only view data
- **Admin Mode:** Can add transactions
- Role switching using dropdown

---

### ## Insights

- Highest spending category
- Monthly comparison
- Total number of transactions

---

### ## UI/UX Enhancements

- Dark mode toggle
- Smooth animations using Framer Motion
- Responsive layout for different screen sizes
- Modern glassmorphism-inspired design

---

### ## Data Persistence

- Transactions are stored in **localStorage**
- Data persists even after page refresh

---

## 🧠 Architecture

The application follows a **component-based architecture**:

- Modular and reusable components
- Centralized state using Context API
- Derived state using `useMemo`
- Separation of concerns:
  - UI components
  - Context logic
  - Utility functions
